Organigramme de la structure du site Franchini

Ce document décrit l'organisation des dossiers et fichiers principaux du projet.

--------------------------------------------------
1. CONFIGURATION DU SITE
--------------------------------------------------
*   `_config.yml` : Fichier de configuration principal de Jekyll. Contient le titre, la description, l'URL du site, et les plugins Jekyll à utiliser.
*   `_data/site.yml` : Fichier de données personnalisé contenant des informations spécifiques au site (coordonnées, réseaux sociaux, ID d'apps, etc.).
*   `Gemfile` : Gère les dépendances de l'environnement Ruby (Jekyll et ses plugins).
*   `package.json` : Gère les dépendances de l'environnement Node.js (utilisé pour le processus de build sur Vercel).
*   `vercel.json` : Configure le déploiement sur la plateforme Vercel.
*   `build.sh` / `vercel-build.sh` : Scripts utilisés par Vercel pour construire le site.

--------------------------------------------------
2. STRUCTURE DES TEMPLATES (Mise en page)
--------------------------------------------------
*   `_layouts/` : Contient les modèles de page principaux.
    *   `default.html` : Le modèle de base pour la plupart des pages du site public.
    *   `admin.html` : Le modèle de base pour le panneau d'administration.
*   `_includes/` : Contient les morceaux de code HTML réutilisables (les "composants").
    *   `header.html` : L'en-tête du site.
    *   `footer.html` : Le pied de page du site.
    *   `head.html` : La section `<head>` du HTML (métadonnées, etc.).

--------------------------------------------------
3. CONTENU ET PAGES
--------------------------------------------------
*   `index.md` : Le fichier de contenu pour la page d'accueil.
*   `pages/` : Contient les autres pages du site (ex: contact, à propos, etc.).
*   `pages/administrateur.html` : Le fichier contenant la structure HTML du panneau d'administration.
*   `_posts/` : (Standard Jekyll) Pour les articles de blog ou actualités.

--------------------------------------------------
4. RESSOURCES (ASSETS)
--------------------------------------------------
*   `assets/` : Dossier principal pour toutes les ressources statiques.
    *   `css/` : Contient les feuilles de style (CSS).
    *   `js/` : Contient les fichiers JavaScript.
        *   `home-carousel.js` : La logique pour le carrousel de la page d'accueil.
        *   `carousel.js` : Une librairie générique pour créer des carrousels.
        *   `auth/firebase-config.js` : Configure la connexion à la base de données Firebase.
        *   `admin/` : Contient tous les scripts spécifiques au panneau d'administration.
            *   `carousel.js` : Gère la logique du carrousel dans l'admin.
            *   `...autres.js` : Logique pour les autres sections de l'admin.
    *   `images/` : Contient les logos, images de fond, etc.
    *   `fonts/` : Polices de caractères personnalisées.

--------------------------------------------------
5. DOSSIER DE BUILD
--------------------------------------------------
*   `_site/` : Ce dossier est généré automatiquement par Jekyll lors de la construction du site. Il contient le site web statique final qui est envoyé à Vercel pour être mis en ligne. **Il ne faut jamais modifier les fichiers dans ce dossier manuellement.**
